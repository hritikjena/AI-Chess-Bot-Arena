# Purpose

The server module is responsible for connecting the frontend interface with the game engine. It handles all API requests and controls the flow of the game.

# Tech stack
	•	Python
	•	Flask (for building API endpoints)
	•	JSON (for data exchange between frontend and backend)

# key workflow

1. create API Endpoint
2. Integrate Game engine(Hritik will make the game engine and provide it, you just to import)
3. Handle Bot execution
4. Manage Game State(Board state, move cout, Turn, result)
5. send response to frontend.

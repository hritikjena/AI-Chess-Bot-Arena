In frontend we have to display the chessboard, animate moves, and communicate with the backend server to fetch game updates.

The interface includes a dynamic chessboard that updates after every move, allowing users to follow the match step by step. It also displays essential information such as current moves, player turns, and final results (win, loss, or draw).

Use:
	•	chessboard.js
	•	HTML + CSS + JS

Responsibilities:
	•	Show chessboard
	•	Animate moves
	•	Call backend API
	•	Display result.
